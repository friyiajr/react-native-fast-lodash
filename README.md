# react-native-fast-lodash

lodash reimplemented in C++ for React Native apps

## Installation

```sh
yarn add react-native-fast-lodash
```

## Usage

```js
import { gcd } from 'react-native-fast-lodash';

// ...

const gcdVal = gcd({
  a: firstVal,
  b: secondVal,
});
```
