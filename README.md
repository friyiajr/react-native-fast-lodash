# react-native-fast-lodash

lodash reimplemented in C++ for React Native apps

## Installation

```sh
npm install react-native-fast-lodash
```

## Usage

```js
import { multiply } from "react-native-fast-lodash";

// ...

const result = await multiply(3, 7);
```
